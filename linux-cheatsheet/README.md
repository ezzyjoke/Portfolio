## File & Directory Commands
`ls`
    What it does: Show files and folders.
    Example: I used it to find files/folders using the terminal

`cd`
    What it does: Move between folders.
    Example: I used it to open folders using the terminal 

`pwd`
    What it does: Show the full path of the folder you are in.
