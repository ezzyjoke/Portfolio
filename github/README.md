# Github formatting cheatsheet
### made by Fadi Abu Mariam

---
## Headers 
"#" H1
"##" H2
"###" H3
"####" H4
#### (biggest to smallest)

---
## **Bold**
"*\*text**"

---
## *Italic*
\*Text*

---
## Links
\[Text]\(Link)

---
## Images
!\[alt text]\(Url)|

---
## code blocks (used to display code without changing it's format)
\```
text
\```
